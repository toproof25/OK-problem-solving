### 입력 

 <p>첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.</p>

### 출력 

 <p>첫째 줄부터 2×N-1번째 줄까지 차례대로 별을 출력한다.</p>

### ***의사 코드***
1. `int star 를 입력받는다`
2. `int repeat = 1`
3. `for (i = 1 ~ 2*star)`
	1. `repeat = (i > star) ? repeat-2 : repeat+2`
	2. `for (blank = abs(star - i) )` - star - i를 절대값을 기준으로 반복하여 공백 출력
		1. `공백을 출력한다`
	3. `for (j = 0 ~ repeat + 1)`
		1. `*을 출력한다`
